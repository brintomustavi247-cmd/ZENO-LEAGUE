## 🎮 Game Economy V6.0 Overview

### Tier System (11 Levels)
```
Level 1: Bronze      (0 XP)      — Entry level
Level 2: Silver      (500 XP)    — Beginner
Level 3: Gold        (1,500 XP)  — Advanced
Level 4: Platinum    (3,500 XP)  — Expert
Level 5: Diamond     (6,500 XP)  — Elite
Level 6: Immortal    (10,500 XP) — Legendary
Level 7-11: Ascended ranks with progressive XP thresholds
```

**Tier Benefits:**
- Color-coded UI (each tier has unique color + glow)
- Perks: commission discounts, entry fee reductions
- Visual badges with hexagon SVG design
- Season leaderboard tracking

### Referral System v2.0
```
User A → Creates referral code → User B joins
                ↓
User B plays 1st match → User A gets 5 TK (locked)
User B plays 3rd match → User A gets 7 TK (locked → unlocked)
User B plays 5th match → User A gets 8 TK (locked → unlocked)
```
**Total possible:** 20 TK locked initially, up to 15 TK unlockable

### Daily Engagement
- **Streaks:** Login rewards, 3x XP on day 7
- **Quests:** 3 daily quests (win, kills, play matches)
- **Ad Rewards:** 5-min cooldown between watches
- **Spin Wheel:** 1 daily spin (in Wallet)

### Social Systems
- **Clans:** Join/create clans, earn collective XP
- **Seasons:** Quarterly rankings with rewards
- **Reputation:** 1-5 star rating based on reviews
- **Leaderboards:** Seasonal rankings
